# EnergiaCar

Descrição:
Aplicação web simples criada para organizar e gerenciar tarefas diárias de funcionários. O sistema exibe atividades do dia, permite marcar checklists e gera uma visualização semanal do que foi executado.

Objetivo:
Testar conceitos de organização de tarefas, interface simples e lógica de controle diário.

Tecnologias:
HTML · CSS · JavaScript
